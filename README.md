# Multilingual and Crosslingual Information System

This repository contains the assignments and projects for the "Multilingual and Crosslingual Information System" course.

> **Special Note on Tech Stack:**
> While the official course curriculum primarily teaches Flutter, the projects in this repository are exclusively built with Native Android (Kotlin). The objective is to fulfill all course requirements and UI designs while practicing modern Android development paradigms.

## Tech Stack & Architecture

This repository serves as a showcase for modern native Android development. Across the assignments, the following core technologies and patterns are applied:

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Architecture:** Clean Architecture (Data / Domain / Presentation layers)
* **Design Pattern:** MVI (Model-View-Intent)
* **Navigation:** Nav3 (Navigation Compose)
* **Networking:** Ktor
* **Dependency Injection:** Koin

## Repository Structure

* **[`/HW1`](./HW1/README.md)**: The first assignment, focusing on building a basic layout (adapted from the course's Flutter tutorial) and implementing a Floating Action Button (FAB) using the modern Android stack mentioned above.
* *(More assignments will be added as the course progresses)*

## Getting Started

To run the projects in this repository:
1. Clone this repository.
2. Open the specific homework folder (e.g., `HW1`) in Android Studio.
3. Sync the Gradle project and run it on an emulator or a physical Android device.
